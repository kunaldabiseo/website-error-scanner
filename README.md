SiteScope is a PHP website error and health scanner: enter a URL and it crawls the site server-side and reports SEO, SSL, security-header, performance, accessibility and WordPress issues, each mapped to severity and owner. PHP must run on a server (GitHub Pages cannot execute PHP): run locally with `php -S localhost:8000` or upload to any PHP host. Requires PHP 7.4+ with curl, dom/xml and mbstring.
